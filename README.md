# webmotely-server

Webmotely server powered by Node.js and WebSockets (proof of concept)

## Setup

Clone this repository
```bash
git clone https://github.com/webmotely/webmotely-server.git && cd webmotely-server
```

Install runtime dependencies
```bash
npm install
```

Start the server
```bash
npm start
```
